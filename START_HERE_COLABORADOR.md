## ⚡ 3 Passos para Começar AGORA

### 1️⃣ SETUP (30 min)

**Opção A - GitHub Codespaces (Recomendado):**
```
1. Vá para: https://github.com/[ORG]/lovable-celite
2. Clique: Code → Codespaces → Create codespace
3. Aguarde 3 minutos
4. Pronto! ✅
```

**Opção B - Na Sua Máquina:**
```bash
git clone https://github.com/[ORG]/lovable-celite.git
cd lovable-celite
pnpm install
# Configure .env.local (peça as credenciais)
pnpm dev
```

### 2️⃣ LEIA (20 min)
1. `INICIO_RAPIDO_COLABORADORES.md` ← Comece aqui!
2. `FRAMEWORK_LOVABLE_CELITE.md` ← Visão geral

### 3️⃣ EXPLORE (1 hora)
- Navegue pelo código em `src/`
- Veja as Edge Functions em `supabase/functions/`
- Rode os testes: `pnpm test`

## 🎯 O Que É Este Projeto?

**Sistema de Comissões MLM para Contadores de Elite**

- **Stack**: Vite + React + TypeScript + Supabase + Stripe
- **Objetivo**: Transformar contadores em consultores de elite
- **Meta emocional**: "Meus filhos terão orgulho de mim"
